# Weather App

Simple web application that shows weather conditions for cities around the world.

## Live Demo

[Live Demo](https://threehundo.github.io/weather-api/)

## Build List

- HTML
- CSS
- JavaScript
- Openweathermap API
- Weather Icons

## Features

Day/Night background synced to current time.
